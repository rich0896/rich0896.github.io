Here are a few HTML elements I have learned so far:

Description Lists
Description lists are a type of list you're seeing right now. I was unaware of their existence before.
Article
An article element houses blog or forum posts, newspaper articles and similar content on a webpage. I was only really aware of divs before, so articles are new for me.
Section
A section element defines a block of information that contains content relevant to eachother. Much like articles, they are a container element I had not witnessed.
Anchor Navigation
While I was aware of anchor elements to hyperlink to other webpages, I was not aware of the ability to target IDs within a webpage.
HTML Entities
While not an element in the same sense as the others, I had not known about entities and character codes, and their existence makes adding special character elements far easier. ⩉∥™⏢
